If you want to upgrade to AIC v1.04, please follow the bellow:
1. Please work it on EFI/UEFI Shell.
2. Change the working directory to this folder.
3. Type UPDboot to upgrade the bootloader to AIC BL v1.00
4. Type UPDapp to upgrade the firmware to AIC v1.04
5. Please power off the machine. And after 5 seconds (or more) , to restart it.
